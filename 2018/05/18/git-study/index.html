2018-05-18
Git

git을 혼자 주먹구구식으로 되는대로 사용하던 것과 비교해, 실제 회사에서 사용하는 것은 commit 한번 하는 게 굉장히 부담스럽고 손 떨리는 일임을 알게 됐다.

그래서 자연스럽게 git을 가장 먼저 공부하게 됐다. 사용방식 자체는 이전과 크게 달라지지 않았지만, git의 원리를 더 깊이 있게 이해하면서 안정적으로 사용할 수 있게 됐다. 물론 난이도 높은 내용들은 아직도 모르지만, 그래도 git과 github의 차이도 잘 이해못하던 때에 비하면 엄청난 발전!

이 과정에서 가장 도움이 됐던 것이 아래 두가지다.
