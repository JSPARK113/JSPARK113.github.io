python 모듈에서 __all__의 쓰임

2018-08-20
python

파이썬 모듈에서 __all__은 외부에서 해당 모듈에 대해 import *를 실행 할 때 export 될 것들을 의미한다.

  • string을 element로 가지는 list다.
  • 모듈의 변수, 함수 모두 export 하는 것이 가능하다.
  • __all__에 지정되지 않은 것들은 *을 해도 외부에서 import되지 않는다.
  • import <함수 이름>을 하면 __all__에 써주지 않아도 부를 수 있다. __all__은 import *에만 영향을 준다.
  • 예시: test.py 파일
    • 아래 모듈에 대해 from test import *을 실행하면 function1과 function2만 import 돼서 사용할 수 있다. function3은 import 되지 않기 때문에 사용할 수 없다.
