파이썬 네임스페이스 패키지(python namespace package)

2018-08-21
python

참고 문서: Packaging namespace packages

보통 하나의 패키지는 하나의 디렉토리 안에서 만들어지는데, 네임스페이스 패키지를 이용하면 여러 디렉토리 안에서 만든 모듈들을 하나의 패키지처럼 같은 이름으로 사용할 수 있도록 만들 수 있다.

네임스페이스 패키지 생성에는 3가지 방법이 있다.

  1. native namespace packages

  2. pkgutil-style namespace packages

    1. 각 패키지 디렉토리마다 __init__.py 파일 생성
    2. __init__.py 파일에 __path__ = __import__('pkgutil').extend_path(__path__, __name__) 코드만 입력한다.
    3. 맨 하위 모듈 파일이 있는 디렉토리의 __init__.py 파일에는 이 내용을 넣지 않아도 된다.

  3. pkg_resources-style namespace packages

참고 문서에서 더 자세한 내용을 볼 수 있다.
