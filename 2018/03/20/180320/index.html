첫번째 팀프로젝트(회귀분석)

2018-03-20

첫 팀프로젝트 진행. 개인적으론 아쉬움이 많이 남은 프로젝트다.

얻은 것

  • 데이터 분석 콘테스트(?)로 유명한 Kaggle을 경험해봤다.
  • 회귀분석의 전체 과정을 어설프게나마 경험해봤다.

아쉬웠던 부분

이 외의 모든 것..?
사실 최선을 다하지 않은 걸 나 스스로 느꼈기 때문에 아쉬움이 남는다. 부족한 부분이 보임에도 그냥 여기까지만 하자 하고 넘어갔는데, 막상 끝나고 나니 조금 더 열심히 해볼 걸 그랬다 싶다. 변명의 여지가 없지만 그래도 조금 해보자면, 회사 다니면서 일에 지나치게 최선을 다했어서 공부하려고 하는 프로젝트엔 그렇게까지 하고 싶지 않은 마음도 좀 있었다.

결론적으론
편하게 하고 나중에 후회하는 것 보다, 열심히 해서 내가 만족스러운 결과를 내는 게 내 성향엔 더 맞는 것 같다.
