블로그 시작! - HEXO로 git 블로그 만들기

공부한 내용을 기록할 겸 블로그를 하나 만들려고 했는데 git 특강 때 들었던 HEXO가 생각나서 도전해봤다.

Django를 이용해서 블로그를 만들 때가 생각나서(…) 어려울 줄 알고 겁먹었는데 시작하는 건 생각보다 간단했다.
블로그에 특화된 거라 그런가 이미 대부분이 만들어져있는 느낌?
마크다운 문서도 익숙하고, 글 작성도 atom에서 할 수 있으니까 혼란스러움이 덜하다.
테마를 바꾸고 좀 꾸며봐야겠지만, 어차피 기록 용도로 쓸거라 거창한 것들이 필요없어서 괜찮을듯.

이 블로그에는 공부한 콘텐츠가 어땠는지 주로 기록할 예정이다.

배운 내용을 자세하게 기록하는 건 너무 큰 작업이라 100% 도중에 포기하게 될 것 같았다.
일단 배운 내용의 양이 너무 많고 그걸 블로그에 올릴만큼 정리하는 게 너무 큰 작업이라서..

책이나 웹 상의 많은 강의, 자료 등을 보고 그게 얼마나 좋았는지, 어떤식으로 도움이 됐는지를 적어놓는 정도로 진행하려고 한다.
(그러려면 일단 많은 것들을 접하고 공부해야겠지만 )

일단 목표는

  1. 읽고, 보고, 들은 콘텐츠들은 모두 기록하기.
  2. 짧은 콘텐츠는 하나의 포스트에 기록, 긴 콘텐츠는 ‘중반(혹은 초반)’, ‘완료 후’ 2개로 포스트에 나눠서 기록.
