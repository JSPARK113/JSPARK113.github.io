awscli 설치해도 aws 명령이 실행 안될 때

2018-05-07
aws

mac OS에 pip install awscli를 통해서 awscli 설치 했음에도
계속 command not found: aws가 뜰 때 해결방법!

내 경우에 아주 간단하게 해결

brew install awscli 로 awscli를 깔자!
(위대한 brew)

시도했으나 효과 없던 방법 - 명령줄 경로에 AWS CLI 실행 파일 추가

aws 사용 설명서에 나와있었으나 이렇게 해도 계속 command not found: aws만 떴음.

이것도 안돼서 역시 위 설명서에 나온대로 python을 3.6 버전으로 깔야하는 건가 고민하던 중에 brew를
통하면 된다는 얘기를 봐서 해봤더니 바로허무하게됨. (위대한 stack overflow)
